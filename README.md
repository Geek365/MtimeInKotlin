# MtimeInKotlin
用现有网络已有时光网API资源，做成一个android客户端，用kotlin+rxjava+retrofit+Material Design风格，主要练手kotlin语法。想看与已看使用本地数据库存储，ROOM库。搜索接口使用豆瓣的，其余使用时光网。
无商业目的，纯练手。。。。
